# 🩺 Diabetes Prediction System

A Machine Learning project for predicting diabetes using the Random Forest algorithm and Streamlit.

---

## 📌 Overview

This project predicts diabetes based on patient health information such as age, BMI, blood glucose level, HbA1c, hypertension history, family history, and lifestyle factors.

The model was built using the Random Forest algorithm and deployed with Streamlit for interactive predictions.

---

## 🚀 Features

- Data preprocessing
- Label Encoding
- Random Forest Classification
- Model Evaluation
- Streamlit Web Application

---

## 🛠️ Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

---
